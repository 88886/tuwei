土味简介

Sign在软件开发中代表着数据签名的意思，目的是为了保护数据安全，从而起到保护知识产权的作用。土味系统使用PHP配合MYSQL开发完成，数据传输使用RSA等加密方式加密保证数据无法被截取，系统主要有软件管理、注册码生成、注册码管理等功能，为软件作者的软件收费管理提供一个绝佳的选择方案。

后台系统介绍

* 以下功能不包括最新功能

商户概况：记录最近的登录时间和IP起到安全提示作用；对所有软件的注册码使用情况进行统计展示。

软件管理：可以新建和修改软件。

  • 试用时间或试用次数设置0代表软件不支持试用；
  • 绑机单开代表注册码只支持一台电脑使用并且每次只能打开一个软件；
  • 绑机多开代表注册码支持一台电脑但是不限制软件打开数量；
  • 不绑机多开代表注册码使用没有任何限制；
  • 允许解绑代表注册码使用者可以在原绑定电脑解绑；
  • 不允许解绑代表只有作者可以解绑；
  • 软件版本和更新地址用于软件强制更新或提示更新；
  • 禁用功能会导致注册码无法登录使用；
  • 生成注册码可以生成任意小时或天数的注册码。
  • 注册码管理可以使用不同筛选条件查找注册码，进行注册码查看、冻结、删除等操作。
  • 批量操作可以批量对注册码进行解绑、冻结、删除操作。
  • 推荐人规则对推荐人和被推荐人进行奖励设置。

    • 关于防破解

    土味插件本身是具有防破解能力的。但是软件的防破解和插件是分开独立的，您的软件集成我们的插件后请一定要进行暗桩加壳等尽可能多的方式增加破解难度。
